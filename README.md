## What I Do

I lead web engineering at Group Link One, where we build **B2B AIoT SaaS platforms** for utilities (water, gas, energy), smart streetlight, out-of-home media, and fraud detection. Five+ products, one multi-tenant architecture, one team that ships.

**Currently:**
- Leading a team of 5 engineers (frontend, backend, fullstack)
- Architected multi-tenant frontend foundation shared across 5+ B2B platforms
- Establishing engineering culture: PR standards, CI/CD, Sentry-driven reliability, AI-assisted workflows

**Previously:**
- 4+ years as Senior iOS Engineer — shipped SDKs, consumer apps, and a **patented IoT communication protocol**
- Built the iOS SDK powering Group Link One's entire AIoT device fleet (BLE/CoreBluetooth/CoreLocation)
- Shipped 6+ production apps (App Store + Enterprise)
